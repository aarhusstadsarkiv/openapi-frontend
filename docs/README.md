## Structure

### openapi.py
This script exposes methods to get api-data as dicts or lists. The methods are as follows:

- Get resource (record, person, collection, vocabulary, user)
  > GET /v1/
- List resources
- create_resource
- batch_create_resources
- get_relation
- list_relations
- post_relation
- batch_create_relation
- autosuggest
- get_user
- create_user
